# инструкция для рабоыт с markdown файлами

## выделения текста
 для выделения текста курсивом нужно выделить текст символом * с обеих сторон
 ```sh
 *text*
 ```

 для выделения текста полужирным выделить текст двойными ** с обеих сторон.

 ```sh
 **text2**
 ```
