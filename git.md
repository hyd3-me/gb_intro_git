# инструкции по работе с git

## создание репозитория:
```sh
git init
```

## добавление файла к отслеживанию
```sh
git add
```

## создание каммита
```sh
git commit -m "msg"
```

## вывод логов каммитов
```sh
git log
git log --oneline
```

## переход на по истории логов
```sh
git checkout
```
